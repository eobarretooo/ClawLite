# Guia de configuração

## Onboarding

```bash
clawlite onboarding
```

Configura:
- modelo padrão
- canais
- token do gateway
- provedor inicial para autenticação

## Menu interativo

```bash
clawlite configure
```

Seções:
- Model
- Channels
- Skills
- Gateway
- Security
- Web Tools

Inclui preview antes de salvar.
