## Por que ClawLite

- **Portável de verdade**: Linux desktop/server e Termux Android
- **Instalação simples**: 1 comando
- **Arquitetura extensível**: skills em Python + registry
- **Gateway nativo**: WebSocket com autenticação por token
- **Pronto para produção incremental**: roadmap de multi-agente e modo offline

## Casos de uso

- Assistente pessoal técnico no terminal
- Motor de automação para bots de mensagem
- Agente local para operações, monitoramento e produtividade
